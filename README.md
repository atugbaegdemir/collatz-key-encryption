# Collatz Varsayımı Temelli Anahtar Üretimi ve Şifreleme

Bu proje, Collatz Varsayımı'na dayalı bir anahtar üretimi ve mesaj şifreleme sistemini göstermektedir.

## Özellikler
- Collatz dizisi kullanılarak anahtar üretimi
- SHA-256 ile güçlendirilmiş anahtarlar
- XOR tabanlı simetrik şifreleme
- Eğitimsel ve deneysel kriptografi projesi
